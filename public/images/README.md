# Imágenes

Coloca aquí el logo de Arándano Café Bar.

## Logo

- **Archivo**: `logo.png` o `logo.jpg`
- **Ubicación**: `/public/images/logo.png`
- **Recomendado**: 
  - Formato: PNG con fondo transparente o JPG
  - Tamaño: Mínimo 800x800px para mejor calidad
  - Peso: Optimizado (< 500KB preferiblemente)

El logo se mostrará automáticamente en la sección Hero de la página.
